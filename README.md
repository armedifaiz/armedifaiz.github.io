# Faiz Armedi — Portfolio

Personal portfolio website — [armedifaiz.github.io](https://armedifaiz.github.io)

Built with React + TypeScript + Vite + Tailwind CSS, deployed to GitHub Pages.

## Quick Start

```bash
npm install
npm run dev        # development server
npm run build      # production build
```

## Data Pipeline

The portfolio fetches external data at build time, not at runtime. There are **zero external API calls** on the client side.

| Script | Source | Fetches |
|---|---|---|
| `fetch-projects.js` | GitHub API (public) | Projects listing from `armedifaiz/*` repos |
| `fetch-credly-badges.js` | Credly (public JSON) | Digital badges from linked accounts |
| `fetch-portfolio-data.js` | `raw.githubusercontent.com` | Photos, certificates, experience, skills, social links |

Run all at once:

```bash
npm run fetch-all
```

### External Assets Repo

Static assets live in a separate public repo — **no rate limits, no API keys**.

```
github.com/armedifaiz/portfolio-data
```

Contents: profile photos, certificate images, JSON metadata (experience, skills, social links).

All fetched via `raw.githubusercontent.com` CDN, cached globally.

## CI/CD

Push to `main` triggers GitHub Actions:

1. Checkout → Setup Node → `npm ci`
2. `fetch-projects.js` → `fetch-credly-badges.js` → `fetch-portfolio-data.js`
3. `npm run build`
4. Upload artifact → Deploy to GitHub Pages
